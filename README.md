# AI Intelligence Hub

An AI-powered news intelligence dashboard that aggregates multi-industry news and generates summaries using OpenAI.

## Live Demo

https://ai-intelligence-app-9bsspaatmxqrky5z3j45xt.streamlit.app

## Features

- Multi-industry news aggregation
- AI-generated summaries
- Category filtering
- Keyword search
- Ask AI about today's news

## Tech Stack

- Python
- Streamlit
- OpenAI API
- RSS feeds
- GitHub + Streamlit Cloud deployment

## Project Structure

app.py – Streamlit application  
news_fetcher.py – RSS news aggregation  
main.py – AI summarization logic  
requirements.txt – dependencies

## Deployment

This project is deployed publicly using Streamlit Community Cloud.
